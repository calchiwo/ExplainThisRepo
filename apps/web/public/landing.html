ExplainThisRepo

An open-source developer CLI tool that explains GitHub repos quickly in plain English

Install using PyPi:
pip install explainthisrepo explainthisrepo owner/repo
Or install using Node:
npm install -g explainthisrepo explainthisrepo owner/repo
